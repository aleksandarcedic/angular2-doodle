creating a new component


  • create new folder under app, name it customer
  • add file customer.component.html
  • add file customer.component.ts
  • in the ts file we need to add
  • import { Component, Input, OnInit } from '@angular/core'; @Component({ moduleId: module.id, selector: 'my-customer', templateUrl: 'customer.component.html' }) export class CustomerComponent implements OnInit { @Input() customer:{id: number, name: string}; constructor() { } ngOnInit() { } }
  • in the html file we need to add
  • 2 spans, 1 for customer.id and one for customer.name
  • in the app.module.ts file we need to add
  • import { CustomerComponent } from './customer/customer.component';
  • and
  • declarations: [ AppComponent,CustomerComponent ],
  • and in our app.component.html we need to call
  • <ul> <li *ngFor="let c of customers"> <my-customer [customer]="c"></my-customer> </li> </ul>
